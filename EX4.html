Voici les 5 grandes étapes d’un projet web :

  1. Comprendre les objectifs du site, les utilisateurs cibles, les fonctionnalités attendues
  2. Planifier la structure et l’apparence du site.
  3. Construire techniquement le site web.
  4. Vérifier que tout fonctionne correctement.
  5. Mettre le site sur un serveur et l’entretenir

outils

  • Barre de navigation (ou menu)
  • Formulaire de contact ou de recherche
  • Fonctionnalités interactives
    • Galerie d'images
    • Carrousel de témoignages
  • Pied de page (footer)
HTML
Langage de balisage utilisé pour structurer le contenu d'une page web.
CSS
Langage de style qui permet de mettre en forme les éléments HTML (couleurs, tailles, disposition, etc.).
JavaScript
Langage de programmation qui permet de rendre les pages web interactives (animations, formulaires dynamiques, etc.).
